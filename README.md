# DevPad

> Bloc de notas flotante para desarrolladores. Simple, rápido y 100% offline.

## ¿Qué es DevPad?

DevPad es una app de escritorio pensada para developers que necesitan un lugar rápido donde anotar ideas, pegar código o armar un README, sin salir de su flujo de trabajo.

La ventana se mantiene siempre visible por encima de otras apps, así podés escribir mientras tenés el editor, la terminal o el navegador abiertos al mismo tiempo.

## Objetivo

Cubrir ese espacio entre "abrir un archivo de texto" y "abrir Notion": algo que arranque al instante, entienda el código y no te pida cuenta, login ni conexión a internet.

## Features planeados

- 📌 Ventana flotante (always on top)
- 🗂️ Múltiples notas con pestañas
- ✍️ Editor con syntax highlighting automático por lenguaje
- 👁️ Preview de Markdown en tiempo real (split view)
- 💾 Guardar y abrir archivos `.md`
- 🌙 Tema oscuro
- 📵 100% offline, sin cuentas ni telemetría

## Stack

- [Tauri](https://tauri.app/) — shell nativo multiplataforma
- [React](https://react.dev/) + TypeScript — interfaz de usuario
- [CodeMirror 6](https://codemirror.net/) — editor de código
- [Marked](https://marked.js.org/) — preview de Markdown
- [Tailwind CSS](https://tailwindcss.com/) — estilos
